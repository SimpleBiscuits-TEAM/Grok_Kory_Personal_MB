  WHAT IS VOP BRIDGE?
  ───────────────────
  VOP Bridge connects your PCAN-USB adapter to the VOP web application
  for real-time vehicle diagnostics, datalogging, ECU scanning, and
  tuning. It runs in the background on your Windows PC.


  FIRST-TIME SETUP (ONE TIME ONLY)
  ─────────────────────────────────
  1. Plug in your PCAN-USB adapter via USB
  2. Double-click "VOP Bridge" on your desktop (or Start Menu)
  3. The bridge will start and show "Ready — waiting for connection"
  4. Open your browser and go to: https://www.ppei.ai
  5. Click "Connect" in the Datalogger — it will find the bridge automatically


  ACCEPTING THE SECURITY CERTIFICATE (ONE TIME ONLY)
  ───────────────────────────────────────────────────
  For secure connections, you need to accept the bridge's certificate:
  1. Open your browser and visit: https://localhost:8766
  2. You'll see a security warning — this is normal
  3. Click "Advanced" then "Proceed to localhost"
  4. You only need to do this once per browser


  DAILY USE
  ─────────
  1. Plug in PCAN-USB adapter
  2. Double-click "VOP Bridge" on your desktop
  3. Open VOP in your browser
  4. Connect and start logging!

  If you enabled auto-start during installation, the bridge starts
  automatically when Windows boots — just open VOP and connect.


  TROUBLESHOOTING
  ───────────────
  • Bridge won't start?
    → Run "Check Status" from Start Menu > VOP Bridge
    → Make sure PEAK drivers are installed (Start Menu > VOP Bridge > Install PEAK Drivers)

  • "No PCAN channel found"?
    → Check USB cable connection
    → Open Device Manager and look for "PEAK-System CAN interfaces"
    → Try unplugging and re-plugging the adapter

  • "Bus error"?
    → Make sure the OBD cable is connected to the vehicle
    → Turn the vehicle ignition to ON (engine can be off)

  • Browser can't connect?
    → Make sure the bridge window shows "WebSocket server listening"
    → Accept the TLS certificate (see above)
    → Try refreshing the VOP page

  • Need to reinstall?
    → Uninstall from Windows Settings > Apps > VOP Bridge
    → Re-run the installer

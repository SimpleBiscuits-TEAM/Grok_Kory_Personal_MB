- MPVI4 (secondary option)

## PPEI-Specific Notes
- Primary device: RTD4
- Secondary device: MPVI4
- Always use latest BETA version of VCM Suite when using laptop
- TDN App is for phone/tablet use with RTD4 or MPVI4
- Invite code changes weekly — customer must contact PPEI for current code
